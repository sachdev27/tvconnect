TvConnect
=========

**TvConnect** is a cross-platform application (built with `BeeWare`_) designed to automate WiFi logins on Android TV and other devices. It highlights features like auto-filling captive portal credentials and a remote control interface for easy text input from your smartphone.

Features
--------

- **Dashboard**: View internet connectivity status and perform quick actions.
- **Auto-Fill & Login**: Automatically injects credentials into captive portal pages (defaults to ACT Corp WiFi).
- **Remote Control**: Starts a local HTTP server and generates a QR code. Scan it with your phone to input username/password remotelyâ€”perfect for typing on TV screens!
- **Settings**: Configure your default WiFi credentials, login URL, and access system settings.

Prerequisites
-------------

- Python 3.8+
- `Briefcase`_
- `toga`
- `pillow`
- `qrcode`

Installation & Running
----------------------

1. Create a virtual environment and install dependencies:

   .. code-block:: bash

       pip install briefcase
       briefcase dev

2. To build for Android:

   .. code-block:: bash

       briefcase create android
       briefcase build android
       briefcase run android
